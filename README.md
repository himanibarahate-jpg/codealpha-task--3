# 🤖 AI Chatbot (Java Swing)

## 📌 Project Name
AI Chatbot using Java

## 📖 Description
This is a simple Java-based AI Chatbot application developed using Java Swing (GUI). The chatbot interacts with users in real-time, takes input messages, and responds using rule-based logic and basic NLP text processing. It is designed as a mini project for learning chatbot development concepts.

## 🎯 Features
- Graphical User Interface (GUI) using Java Swing  
- Real-time chat interaction  
- Rule-based response system (FAQ matching)  
- Basic NLP text processing (input cleaning)  
- Keyword-based responses  
- Lightweight and easy to run  

## 🛠 Technologies Used
- Java  
- Java Swing (GUI)  
- HashMap (for storing responses)  
- Basic NLP (text preprocessing)

## 📂 File Structure
ChatBot.java   → Main file (run this file)

## ▶ How to Run

Compile:
javac ChatBot.java

Run:
java ChatBot

## 💬 How It Works
1. User types a message in the input box  
2. Program converts text to lowercase and removes special characters  
3. Bot checks keywords using predefined rules (FAQ)  
4. Matching response is returned  
5. Conversation is displayed in GUI chat window  

## 💡 Example Output
You: hello  
Bot: Hi! How can I help you?

You: what is java  
Bot: Java is a programming language.

You: bye  
Bot: Goodbye! Have a nice day.
